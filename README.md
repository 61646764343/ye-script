✨ 项目简介

夜脚本（Ye Script） 是一套专为 Roblox 平台 打造的 多功能脚本合集，集成：

- 🎮 25+ 热门游戏专用脚本
- 🔧 强大的通用功能模块
- ⚡ FE（过滤开启）兼容脚本
- 🎨 现代化 UI + 主题系统
- 💾 配置自动保存

📌 目标：稳定 · 易用 · 功能全面

🚀 一键加载（推荐）

使用任意支持 
"loadstring" 的 Roblox 执行器
-- 📥 主加载脚本
loadstring(game:HttpGet("https://raw.githubusercontent.com/61646764343/ye-script/refs/heads/main/ye%20script.lua", true))()
