✨ 项目简介

夜脚本（Ye Script） 是一套专为Roblox平台开发的多功能脚本集合，集成了超过25款热门游戏的自动化脚本、通用功能模块和强大的FE工具。本脚本采用模块化设计，拥有美观的图形界面，支持主题自定义和配置保存，为玩家提供稳定、高效的游戏增强体验。

🎯 核心特色

· 🎮 游戏全覆盖：支持DOORS、忍者传奇、极速传奇、鲨口求生2、彩虹朋友等25+热门Roblox游戏
· 🔧 功能多元化：集成了玩家属性修改、穿墙、夜视、飞行、无敌等通用功能
· 🎨 界面美观：采用现代化UI设计，支持主题切换和DPI缩放
· 💾 配置保存：自动保存用户设置，支持主题管理和快捷键绑定
· ⚡ FE兼容：包含大量过滤开启(FE)脚本，可在受限制服务器中安全使用

🚀 快速开始

一键加载脚本

使用你的Roblox执行器，在游戏内执行以下代码：

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/61646764343/ye-script/refs/heads/main/ye%20script.lua", true))()
```

🚨 常见问题

Q: 脚本加载失败怎么办？

A:

1. 检查网络连接是否正常
2. 确保执行器支持HTTPS请求
3. 尝试使用备用加载方式
4. 查看控制台错误信息

Q: 功能无效或无法使用？

A:

1. 确认当前游戏被脚本支持
2. 检查游戏是否更新导致脚本失效
3. 尝试重启游戏和脚本
4. 在QQ群反馈问题

Q: 如何添加自定义脚本？

A:
目前不支持直接添加，但可以通过修改源码或联系作者添加。
